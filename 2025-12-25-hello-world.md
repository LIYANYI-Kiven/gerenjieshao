## 如何添加新文章？

非常简单！你只需要在 `_posts` 文件夹中创建一个新的 Markdown 文件。

文件名的格式必须是：`YYYY-MM-DD-标题.md`，例如 `2025-12-26-my-new-post.md`。

文件的开头需要包含以下“Front Matter”：

```yaml
---
layout: post
title:  "文章标题"
date:   2025-12-26 12:00:00 +0800
categories: jekyll update
---
```

然后，你就可以使用 Markdown 语法撰写正文了。

## Markdown 语法示例

你可以轻松地添加：

- **粗体文字**
- *斜体文字*
- [链接](https://jekyllrb.com)
- 列表

甚至代码块：

```javascript
console.log('Hello, World!');
```

当你把文件推送到 GitHub 后，网站会自动更新！
